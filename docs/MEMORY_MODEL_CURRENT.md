# U-lang Memory Model — Реальное состояние

## Что работает сейчас

### 1. Создание структур ✅
```u
x = Counter(value: 0)   # создали, x — владелец
```

### 2. Мутация полей ✅
```u
x.value = 5             # изменили поле
```

### 3. Присваивание (Copy) ✅
```u
y = x                   # y получает копию данных
```

### 4. Функции получают копию ✅
```u
fn consume(d: Data)     # d — копия, оригинал жив
```

## Что НЕ работает

### Move семантика 🚧
```u
consume(x)              # x всё ещё доступен после вызова
                        # (move не реализован)
```

### References/borrowing 🚧
```u
ref = x.value           # нет ссылок, только копии
```

## Текущая модель

**Copy-semantics для всего.** Все присваивания — копии.

## Пример

```u
struct Counter
    value: Int
end

x = Counter(value: 0)
x.value = 5             # ✅ мутация
y = x                   # ✅ копия
y.value = 10            # ✅ мутация копии
```
