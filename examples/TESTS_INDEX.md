## Ожидаемые проблемы

### Проблема 1: Clone для канала в spawn
В текущей реализации `ch` клонируется через `.clone()`, но кортеж `(Sender, Receiver)` 
не реализует `Clone` для `Receiver`. Нужно передавать только `Sender` в spawn.

**Решение:**
```rust
// До (не работает)
let ch = ch.clone();  // Receiver не Clone

// После (работает)
let tx = ch.0.clone();  // только Sender
```

### Проблема 2: Типизация каналов
Генератор не знает тип T для канала. Нужен механизм вывода типа из первого send.

### Проблема 3: Blocking receive
`.receive()` блокирует — нужен async runtime.
