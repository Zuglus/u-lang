# CLAUDE.md — контекст для Claude Code

## Проект

U Language — единый язык программирования. Транспилятор: U syntax → Rust code → rustc.

## Спецификация

Полная спецификация в `spec/SPEC.md`. Примеры в `examples/`.

## Ключевые решения

- **Два режима**: `memory(auto)` (RC+GC, горутины) и `memory(own)` (ownership, async)
- **`::` для мутации**: `.` — чтение, `::` — мутация (методы, поля, аргументы)
- **Скрипт по умолчанию**: нет `memory(...)` → всё автоматически
- **`end`-блоки**: вместо `{}`
- **4 модификатора**: `unsafe`, `weak`, `test`, `public`
- **Лямбда**: одно выражение: `fn(x) x > 0`
- **`?`**: пробрасывание ошибок и nullable
- **`$`**: строковая интерполяция
- **`use`**: обязателен только в проектном режиме (с `memory(...)`)
- **Return**: явный. Нет return = процедура.
- **Типы**: `public` fn — с типами, остальные — без
- **Скобки `()`**: обязательны при вызове функции

## Архитектура транспилятора

```
.u файл → [Parser] → AST → [Generator] → .rs файл → rustc → бинарник
```

Транспилятор на Rust. Парсер: pest. CLI: clap.

## Команды

```
cd transpiler
cargo build          # собрать транспилятор
cargo test           # запустить тесты
cargo run -- run examples/hello.u    # запустить .u файл (когда готов)
```

## Приоритет работы

1. Парсер: разбор .u файла в AST
2. Генератор: AST → Rust-код
3. CLI обёртка: `u build`, `u run`
4. Маппинг ошибок: rustc ошибки → .u файл
5. Рантайм: crate для memory(auto) (RC+GC, горутины)
